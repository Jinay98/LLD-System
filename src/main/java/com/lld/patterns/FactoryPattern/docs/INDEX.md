# Factory Pattern - Object Creation

Complete guide to the Factory Pattern and its variants.

## 📚 Files
- **README.md** - Factory Pattern theory and examples

## 🎯 When to Use Factory
- Multiple types to create
- Type determined at runtime
- Creation logic is complex
- Want to decouple creation from usage

## 🔄 Factory Variants
1. Simple Factory (static method)
2. Factory Method (inheritance-based)
3. Abstract Factory (family of factories)

## 🔑 Key Takeaway
**Factory Pattern:** Encapsulate object creation, decouple from usage
